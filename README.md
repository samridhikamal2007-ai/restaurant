# 🍴 Restaurant Website

## 🚀 Overview
This project is a simple restaurant website designed to showcase menu items, ambiance, and contact information.  
It demonstrates core web development skills including HTML, CSS, and responsive design.

## ✨ Features
- 📋 Menu section with food categories
- 🖼️ Image gallery for dishes and interiors
- 📍 Location and contact details
- 📱 Responsive layout for mobile and desktop
- 🎨 Custom styling for a unique brand identity

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (optional for interactivity)
- GitHub Pages (for hosting)
